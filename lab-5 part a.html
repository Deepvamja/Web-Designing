Concept of WWW

  • it stands for world wibe web.
  • it is basically a system of internet server that support specially formatted documents.
  • it is a collection of text pages,videos,digital photographs and animations we can access over the internet.
  • User access the World Wibe Web facilities via client called a web browser.

HTTP Protocol

  1. HTTP stands for Hypertext Transfer protocol.
  2. HTTP is based on the client-server architecture model and a stateless request/response protocol.
  3. 404 File Not Found is a common HTTP status code.
